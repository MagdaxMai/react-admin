# react-admin

## Developoment

### Requirements

-   [yarn](https://yarnpkg.com/)
-   [watchman](https://facebook.github.io/watchman/)
-   [wml](https://github.com/wix/wml)

### Run build

    yarn install
    yarn build

### Run watcher

    yarn start

### Develop in an project

    ./wml-add.sh ../example
    wml start
